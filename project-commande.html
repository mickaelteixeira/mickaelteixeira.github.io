Commande.com Refonte

Développeur

Mission & Réalisation

Refonte complète de la plateforme e-commerce basée sur Sylius.

L'objectif était d'intégrer un moteur e-commerce headless et de développer des fonctionnalités métiers spécifiques (logistique, promo, export), tout en gérant des flux interconnectés complexes.

Réalisations Clés

  • Intégration d’un moteur e-commerce headless.
  • Gestion de flux interconnectés (ERP, transporteurs, outils internes).
  • Mise en place de tests automatisés (PHPUnit, Behat) et qualité de code (PHPStan, CS Fixer).

La Stack

PHP 8.2 Symfony Sylius API Platform Docker RabbitMQ Elasticsearch GitLab CI
